# Robot Naming Convention

K-Scale robot names should follow the convention `<application>-<series>`.

## Series Designations

- **D**: Domestic. This designation is for robots that are intended to operate in a home environment, such as cleaning, assistance, and other domestic tasks.
- **P**: Prototyping. This designation is for robots that are intended to be used primarily for prototyping and experimentation, without having immediate commercial applications.
- **I**: Industrial. This designation is for robots that are indended to be used in industrial settings, such as manufacturing, assembly and other tasks.
