# K-Brain

<Callout type="warning" emoji="⚠️">
Note that you should do `pip install k-brain` instead of `pip install kbrain`, but do `import kbrain` in your code.
</Callout>

```bash
pip install k-brain
```

K-Brain combines the hardware, software and firmware to provide an open-source "skateboard" for humanoid robots.
